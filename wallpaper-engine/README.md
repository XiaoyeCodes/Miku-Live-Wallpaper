# 初音未来动态壁纸（Wallpaper Engine Web）

## 功能

- 根据电脑本地时间自动切换晨曦、早晨、上午、中午、下午、傍晚和晚上场景。
- 场景到达时间锚点后，在 5 秒内通过透明度平滑交叉淡化。
- 透明数字时钟支持日期、秒钟、大小、垂直位置、分时颜色和自定义颜色。
- 音乐频谱可在环形与横向两种模式间切换。
- 环形频谱支持位置、大小、半径、条数、条宽、幅度、四色渐变、旋转和中心图片。
- 横向频谱支持位置、宽度、幅度、条数、间距、方向以及 1 至 4 种自定义颜色。
- 中心圆盘使用动态低频门槛、上升沿和冷却时间检测鼓点，并以短促弹簧动画回弹。
- 音频采样与画面帧率分离，画面可选择 30、60、90、120、144 或 240 FPS。
- 可选 ASUS Aura COM 桥接，将音乐能量同步到兼容灯效设备。

## 时间锚点

| 场景 | 开始时间 |
| --- | --- |
| 晨曦 | 05:00 |
| 早晨 | 07:00 |
| 上午 | 09:00 |
| 中午 | 12:00 |
| 下午 | 14:00 |
| 傍晚 | 17:30 |
| 晚上 | 20:00 |

可在 `wallpaper.js` 顶部的 `scenes` 数组中修改时间和图片路径。

## 导入 Wallpaper Engine

1. 打开 Wallpaper Engine，点击 **创建壁纸**。
2. 选择网页壁纸并导入本目录中的 `index.html`。
3. 保存项目并重新载入一次，使 `project.json` 中的中文属性和音频处理配置生效。
4. 在右侧属性面板调整频谱、鼓点、中心图片和时钟。

项目已经包含维护好的 `project.json`，请勿用空白配置覆盖它。Wallpaper Engine 音频接口会提供 128 个频段值；普通浏览器预览时，项目会自动使用模拟音乐数据，Wallpaper Engine 内则使用真实系统音频。

## ASUS Aura Sync（可选）

Aura 功能通过本地 Python 桥接程序调用已注册的 Aura SDK v3.1 COM 接口：

1. 安装 64 位 Python、`pywin32`、Armoury Crate/Aura SDK 组件。
2. 关闭可能独占灯效控制的 Opera GX Lights。
3. 双击 `start-aura-bridge.bat`。
4. 保持桥接窗口运行并在 Wallpaper Engine 中播放音乐。

桥接仅监听 `127.0.0.1:27340`，不接受远程连接。若 Aura SDK 无法枚举设备，灯效不会生效；详细说明见 `AURA_BRIDGE.md`。
